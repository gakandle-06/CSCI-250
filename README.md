# CSCI-250
## Programming for Data Application

### Course Overview
This repository contains projects focused on data-oriented Python programming and foundational Machine Learning algorithms. The course emphasizes practical implementation of predictive models and data analysis techniques.

### Key Technologies & Packages
* **Language:** Python 3.x
* **Data Manipulation:** `pandas`, `NumPy`
* **Visualization:** `matplotlib`, `seabor`
* **Machine Learning:** `scikit-learn`

### Machine Learning Concepts Explored
This project implements and explores several core ML architectures:
* **Decision Trees:** Building interpretable models for classification and regression.
* **Support Vector Machines (SVM):** Implementing linear and non-linear boundaries for data separation.
* **Neural Networks:** Understanding basic multi-layer perceptron structures.
* **Classification & Regression:** Applying various supervised learning techniques to real-world datasets.
